# shp-shop
## 一个基于vue全家桶开发的电商平台(尚硅谷的商品汇电商前台项目)
### 技术栈：Vue2 Vuex Vue-router axios
### 核心模块：登录、注册、商品列表、商品详情、搜索、购物车、订单、支付
